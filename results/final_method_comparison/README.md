# Final Method Comparison Folder

This folder contains the clean final comparison outputs for the invoice extraction project.

Use `final_comparison_summary.md` for the report explanation, and use the CSV files for tables in the final thesis/project documentation.

Main result:

| Method | Overall Accuracy |
|---|---:|
| OCR + Regex | 33.33% |
| Fine-tuned LayoutLMv3 | 58.91% |
| Proposed Layout-Aware Method | 75.97% |

The proposed layout-aware method is the best-performing method on the same 20-document evaluation split.
